# React Movie DB Viewer

## Running The App

Within the app directory, type in:

* `npm install`
* `npm start`

## Challenge Overview
* List popular movies and search bar on initial load
* Search should list movies pertaining to query
* Clicking on a movie should route to details page

## Component Hierarchy
* App
  * NavBar
    * SearchBar
    * DropDown
  * MovieIndex
